Employee User Guide

Function Search

the search function is a field that allow you to find employees just typing the name or id like image below.



the option "include removed employee" , will include inactives employee on search.

Employee add / view / edit / remove



To register a new employee use the button Add Employee and you will be redirected to the page to register a new employee.
To view employee informations use the button and you will be redirected to the page of the employee selected .
To update employee informations use the button and you will be redirected to the page of the employee selected , this link will also allow you to add employee to the project and mentee to the Mentor.
To remove employee use the button after insert the leaving date at the popup like image bellow.
